# 智选商城模拟物流政策

> 说明：本政策为学习 Agent 工作流而设计的模拟规则，参考公开常见电商售后流程后，为 Agent 学习与评测设计，不代表真实平台政策、法律意见或消费者权益承诺。

## 适用范围

本政策适用于智选商城模拟订单的发货、运输、签收、拒收和物流异常咨询。当前项目只保存模拟物流状态，不接入真实快递接口。

## 标准流程

支付成功后订单进入已付款状态，通常在 48 小时内安排发货。发货后订单状态为 `shipped`，表示商品已进入物流履约环节。签收后订单状态为 `delivered`，并记录 `delivery_time`。已发货未签收订单不能按未发货订单直接取消，可建议用户关注物流、选择拒收或签收后申请售后。

## 例外与人工处理情形

物流长时间未更新、包裹疑似丢失、用户已拒收但系统未更新、签收人与用户不一致、物流状态和订单状态冲突等情况需要人工客服核实。发货拦截、退回仓库、运费争议等流程当前不由规则引擎自动处理。

## Agent 可执行边界

Agent 可以解释 `unshipped`、`shipped`、`delivered` 等模拟物流状态，并给出拒收、等待签收后售后或转人工建议。Agent 不能真实拦截快递、不能修改收货地址、不能联系快递员、不能改变真实物流状态。缺少物流状态、签收时间或订单号时，应提示补充信息。
