# 智选商城模拟服务范围政策

> 说明：本政策为学习 Agent 工作流而设计的模拟规则，参考公开常见电商售后流程后，为 Agent 学习与评测设计，不代表真实平台政策、法律意见或消费者权益承诺。

## 适用范围

本政策说明智选商城模拟售后 Agent 当前可处理和不可处理的范围，适用于学习、测试、评测和演示场景。

## 标准流程

当前 Agent 可处理政策咨询、订单状态查询、退款资格初步判断和模拟工单创建。政策检索层可以根据用户问题召回相关模拟政策片段。订单 Tool 可以查询本地 SQLite 模拟订单。退款资格 Tool 可以基于确定性规则返回可解释结论。模拟售后助手不能帮用户执行真实退款，只能给出政策解释、资格初判和转人工建议。

## 例外与人工处理情形

复杂或超出规则的问题进入 `manual_review` 或建议转人工。高风险支付问题、账户安全、隐私泄露、质量争议、物流争议、优惠券争议、库存换货争议等不应由 Agent 自动给最终承诺。

## Agent 可执行边界

Agent 不可直接执行真实退款、真实取消订单、修改地址、支付操作、发票开具、积分调整或真实客服转接。所有数据均为模拟数据。无关问题不应强行匹配政策，应返回未检索到足够相关政策内容，并建议补充问题或转人工。
