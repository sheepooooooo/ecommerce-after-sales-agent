# 智选商城模拟会员积分政策

> 说明：本政策为学习 Agent 工作流而设计的模拟规则，参考公开常见电商售后流程后，为 Agent 学习与评测设计，不代表真实平台政策、法律意见或消费者权益承诺。

## 适用范围

本政策适用于智选商城模拟订单中的会员积分获得、退款后的积分扣回、积分异常和积分边界说明。

## 标准流程

订单完成后可获得模拟积分。退款成功后，对应积分会被扣回。积分不可直接兑换现金，也不能替代真实支付金额。积分规则以模拟订单状态为基础，不接入真实会员账户系统。

## 例外与人工处理情形

积分未到账、积分扣回异常、用户对积分数量有争议、订单状态与积分状态不一致时，需要人工核实。本阶段不处理真实积分账户、积分商城兑换或积分有效期。

## Agent 可执行边界

Agent 可以解释积分获得和退款扣回的模拟规则，可以建议用户提供订单号和积分异常描述。Agent 不能真实发放积分、不能修改积分余额、不能兑换现金。
