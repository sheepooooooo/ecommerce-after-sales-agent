# 智选商城模拟优惠券政策

> 说明：本政策为学习 Agent 工作流而设计的模拟规则，参考公开常见电商售后流程后，为 Agent 学习与评测设计，不代表真实平台政策、法律意见或消费者权益承诺。

## 适用范围

本政策适用于智选商城模拟订单中的优惠券使用、订单取消后的优惠券返还、退货退款后的优惠券争议和优惠券限制说明。

## 标准流程

优惠券不可兑换现金。每个订单最多使用一张普通优惠券。订单取消且未发货时，如果优惠券仍在有效期内，可按模拟系统规则退回。已发货后发生退货退款，优惠券是否返还取决于订单完成状态、退款范围和券有效期。

## 例外与人工处理情形

优惠券过期、订单部分退款、拆单、组合优惠、优惠券返还失败、用户要求现金补偿等情况需要人工核实。本项目当前退款规则引擎不计算优惠券，也不处理复杂券规则。

## Agent 可执行边界

Agent 可以解释优惠券不等同现金、可能按模拟规则返还以及争议需人工核实。Agent 不能真实发券、不能延长优惠券有效期、不能把优惠券兑换为现金、不能修改订单优惠金额。
