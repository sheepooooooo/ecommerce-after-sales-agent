# 智选商城模拟换货政策

> 说明：本政策为学习 Agent 工作流而设计的模拟规则，参考公开常见电商售后流程后，为 Agent 学习与评测设计，不代表真实平台政策、法律意见或消费者权益承诺。

## 适用范围

本政策适用于智选商城模拟订单中的换货咨询，包括质量问题换货、非质量问题换货、数码商品拆封后的换货限制和缺货处理。

## 标准流程

质量问题商品在签收后 15 个自然日内可申请换货。非质量问题换货需要在签收后 7 个自然日内提出，且商品未拆封、包装和配件完整。换货商品应与原订单商品保持一致或由人工确认可替换方案。商品缺货时，用户可选择等待补货或改走退款流程。

## 例外与人工处理情形

已拆封且无质量问题的数码商品不支持自动换货。库存不足、用户要求更换不同型号、配件缺失、质量争议、物流破损责任不清等情况需要人工确认。换货后的物流安排、补发时效和库存锁定当前不由系统自动处理。

## Agent 可执行边界

Agent 可以根据签收天数、质量问题和拆封状态给出换货建议。Agent 不能真实锁库存、不能创建真实换货单、不能安排真实物流。缺少签收时间、商品状态、质量问题描述或库存信息时，应建议转人工。
