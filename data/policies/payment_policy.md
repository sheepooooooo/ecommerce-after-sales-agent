# 智选商城模拟支付政策

> 说明：本政策为学习 Agent 工作流而设计的模拟规则，参考公开常见电商售后流程后，为 Agent 学习与评测设计，不代表真实平台政策、法律意见或消费者权益承诺。

## 适用范围

本政策适用于智选商城模拟订单中的支付状态、支付异常、重复扣款、退款路径和支付争议咨询。

## 标准流程

智选商城支持模拟支付宝、微信支付和银行卡支付。未付款订单不产生退款。支付成功后订单进入已付款状态，后续才可能进入发货、取消或退款流程。退款默认按实际支付金额原路返回。

## 例外与人工处理情形

支付扣款成功但订单状态未更新时，例如用户已经付款但订单仍显示待付款、支付成功后页面一直没有刷新、支付成功但订单没有进入已付款状态，需要人工核实支付流水和订单状态。重复扣款、支付金额异常、退款长时间不到账、支付账户安全风险等问题建议创建工单转人工处理。

## Agent 可执行边界

Agent 可以说明未付款不退款、支付成功后订单才进入已付款状态、退款默认原路返回。Agent 不能查询真实支付流水、不能发起真实扣款或退款、不能要求用户提供银行卡密码、支付密码或验证码。
